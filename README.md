# 🚀 Landing Page - Kumpulan Tools Web

Selamat datang di repository **Landing-page**. Proyek ini merupakan kumpulan dari berbagai macam aplikasi berbasis web (HTML/CSS/JS) yang dirancang untuk memudahkan berbagai keperluan administrasi, verifikasi, dan pengolahan data.

## 📖 Tentang Proyek Ini

Repository ini berfungsi sebagai pusat (*central hub*) dari berbagai tools ringan yang dapat dijalankan langsung di browser tanpa perlu instalasi. Setiap tool dikemas dalam folder atau file tersendiri agar mudah dikelola dan diakses.

## 🛠 Daftar Tools

Berikut adalah daftar tools yang tersedia beserta fungsinya masing-masing:

| No | Nama Tool | Deskripsi Singkat | Akses (Folder/File) |
| :---: | :--- | :--- | :--- |
| 1 | **Landing Page Utama** | Halaman pembuka untuk mengakses semua tools. | `/index.html` |
| 2 | **Aplikasi Scan Dokumen Web** | Memudahkan proses scan dan konversi dokumen via web. | *(Sesuaikan dengan nama folder)* |
| 3 | **EXIF Explorer** | Membaca dan menampilkan metadata (EXIF) dari file foto. | *(Sesuaikan dengan nama folder)* |
| 4 | **Form Verifikasi (Kesos)** | Form untuk verifikasi data sosial/administrasi kependudukan. | *(Sesuaikan dengan nama folder)* |
| 5 | **Generator Daftar Hadir P2K2** | Membuat daftar hadir otomatis untuk kegiatan P2K2. | *(Sesuaikan dengan nama folder)* |
| 6 | **Image Resizer Pro** | Mengubah ukuran gambar (resize) secara praktis. | *(Sesuaikan dengan nama folder)* |
| 7 | **Kalkulator PKH Modern** | Kalkulator khusus untuk menghitung bantuan Program Keluarga Harapan (PKH). | *(Sesuaikan dengan nama folder)* |

> **Catatan:** Jika ada tools lain yang belum tercantum di tabel, silakan tambahkan baris baru sesuai dengan struktur folder yang ada.
